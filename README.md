# Smart Meal Finder
Okos társ az étkezésben<br>
A projekt célja, hogy segítse a felhasználókat receptek megtalálásában saját preferenciáik alapján, ezáltal támogatva az egészséges étkezés fenntartását.

<ul>
  <b>Főbb funkciói:</b>
<li>Hozzávaló alapú receptkeresés</li>
<li>Felhasználói profil létrehozása:</li>
Étkezési korlátozások,
Fitneszcélok beállítása,
Célok felkövetése.
<li>Receptek ajánlása az étkezési profil alapján</li>
<li>Heti étkezési terv összeállítása és ehhez bevásárlólista generálás</li>
</ul>

> [!IMPORTANT]  <p name="important"></p> 
> Első futtatáskor a fő mappában található docker compose-t futtasd a "docker-compose up --build" paranccal!
